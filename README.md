# gitlab-watcher

simple app to watching and notify gitlab issues event, based on GitHub's Electron.

------

## Install

* Install electron

```bash
$ npm install electron-prebuilt -g
```

* Clone repo

```sh
git clone git@github.com:lisposter/gitlab-watcher.git
```

* Install deps

```sh
$ npm i
$ bower i
```

* Launch

```sh
$ electron /path/to/repo
```
